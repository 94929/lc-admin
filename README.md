# lunchat_admin
<br>

---

### 1. 의존성 설치
```

yarn

```

---

### 2. 로컬서버 실행
```

yarn dev

```

---

### 3. 빌드

```

yarn build

```
---

### 4. 프로덕션 실행 (빌드 후 실행)
``` 

yanr start

```

<br>

### libs<br>

개발환경 : [NextJs_v13](https://nextjs.org/)
<br>
UI 라이브러리 : [Antd](https://ant.design/components/overview/)
<br>
CSS 라이브러리 : [tailwind](https://tailwindcss.com/)
<br>
상태관리 라이브러리 : [Zustand](https://zustand-demo.pmnd.rs/)
